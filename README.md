# ClickDefense

Mobile-first Vite + React wave defense clicker.

Players tap a moving target to break waves, survive shield damage, earn coins,
and buy persistent upgrades between runs.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- No backend is required for the MVP.
- Local progress is stored in `localStorage`.
- Desktop uses a side-by-side game and workshop layout.
- Mobile uses a focused game screen, then a between-runs workshop screen.
